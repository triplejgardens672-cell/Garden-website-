# Triple J Gardens - Website

A professional website for Triple J Gardens, showcasing garden design, landscaping, and maintenance services.

## Project Structure

```
Garden-website-/
├── index.html          # Home page
├── services.html       # Services page
├── about.html          # About us page
├── contact.html        # Contact page
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # JavaScript functionality
├── images/             # Image assets
└── README.md           # This file
```

## Features

- Responsive design (mobile, tablet, desktop)
- Modern, clean aesthetic
- Service showcase
- Contact form
- Gallery section
- Company information

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. Customize content with your company information
4. Add your own images to the `images/` folder
5. Update colors and branding in `css/style.css`

## Pages

- **Home** - Landing page with hero section and featured services
- **Services** - Detailed list of garden and landscaping services
- **About** - Company history and team information
- **Contact** - Contact form and company details

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
